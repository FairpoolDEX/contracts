# Shield Finance contracts

## Setup

    # Install NVM: https://github.com/nvm-sh/nvm#installing-and-updating
    nvm install $(cat .nvmrc)
    nvm use # loads version from .nvmrc
    npm install
    npm run build

## Tasks

Build the project

    npm run build

Run tests

    npm test

Watch tests

    npm run test:watch

Lint code

    npm run lint

Generate coverage

    npm run coverage
